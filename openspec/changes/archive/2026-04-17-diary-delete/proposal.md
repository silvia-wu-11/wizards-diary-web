# Proposal: 日记批量删除功能

## 背景 (Background)
目前魔法日记本应用中，用户可以创建、查看和筛选日记，但缺少删除日记的功能。为了让用户能够管理自己的日记，我们需要提供一种直观、便捷的方式来删除不需要的日记记录。

## 目标 (Goals)
1. 在首页日记筛选搜索栏右侧增加“删除”入口。
2. 支持批量选择日记并进行删除。
3. 从 UI 和数据库（Supabase PostgreSQL）中彻底删除选中的日记。
4. 在操作成功或失败时给予用户明确的提示。

## 非目标 (Non-Goals)
1. 暂不支持日记回收站（软删除），本次实现为硬删除（直接从数据库中删除）。
2. 暂不在日记详情页内增加单独的删除按钮，本次仅在首页实现批量删除交互。
